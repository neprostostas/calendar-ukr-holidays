# Календар Українських Свят

Інтерактивний односторінковий вебсайт — календар національних свят та офіційних вихідних України.  
Створений у стилі **магічного фоліанту мольфарів**: зоряне небо, місячне світіння, золоті акценти та українська міфологічна атмосфера.

**Автор:** Liubomur Kinash

## Про проєкт

Сайт задуманий для щоденного використання українцями: швидко побачити поточну дату, знайти найближче свято, дізнатися скільки днів залишилось до вихідного, переглянути календар місяця з позначками державних свят.

Замість звичайної сітки дат — **«Книга знань»**: запис дня, чарівні символи біля свят, модальне вікно з коротким описом традицій при натисканні на день.

---

## Можливості

- Календар поточного місяця з навігацією вперед / назад
- Відображення сьогоднішньої дати українською
- Підсвітка поточного дня
- Позначення **державних свят** (символ ✦)
- Позначення **офіційних вихідних** (субота та неділя)
- Блок **«Найближче свято»**
- Блок **«Скільки днів залишилось»** — зворотний відлік
- Модальне вікно з назвою, датою та описом свята
- Адаптивний дизайн для мобільних і десктопних екранів
- Декоративні ефекти: зірки, паралакс, мерехтливі частинки, місяць, силует лісу

## Структура файлів

```
calendar-ukr-holidays/
├── index.html          # Розмітка сторінки, SEO-мета, підключення стилів і скрипта
├── style.css           # Візуальний стиль, тема, адаптивність, анімації
├── script.js           # Логіка календаря, дані свят, інтерактивність
├── favicons/           # Іконки сайту та webmanifest
└── README.md
```

### `index.html`
Головна сторінка. Містить:
- фон (небо, зірки, місяць, частинки);
- шапку-фоліант з назвою проєкту;
- блок запису поточного дня;
- інформаційні картки;
- сітку календаря;
- модальне вікно для опису свята.

### `style.css`
Відповідає за всю візуальну частину:
- кольорова палітра (темно-синій, фіолетовий, золото, місячне світіння);
- стилізація карток, клітинок календаря, модального вікна;
- паралакс зірок і анімації;
- медіа-запити для різних розмірів екрана.

### `script.js`
Містить усю логіку:
- масив `HOLIDAYS` — державні свята України (назва, дата, опис);
- `MOVABLE_HOLIDAYS` — дати Великодня та Трійці по роках;
- побудова календаря на обраний місяць;
- пошук найближчого майбутнього свята;
- відкриття модального вікна при кліку на день зі святом.

Дані зберігаються локально — **без API і серверної частини**.

## Свята в календарі

| Свято | Дата |
|---|---|
| Новий рік | 1 січня |
| Різдво (православне) | 7 січня |
| Міжнародний жіночий день | 8 березня |
| Великдень | рухома дата |
| День праці | 1 травня |
| День перемоги над нацизмом | 9 травня |
| Трійця | рухома дата |
| День Конституції України | 28 червня |
| День Незалежності України | 24 серпня |
| День захисників і захисниць України | 1 жовтня |
| Різдво (католицьке) | 25 грудня |

Щоб додати нове свято — доповніть масив `HOLIDAYS` у `script.js`.  
Для рухомих свят (Великдень, Трійця) — оновіть об'єкт `MOVABLE_HOLIDAYS`.

## Як запустити

Проєкт не потребує встановлення залежностей чи збірки.

**Варіант 1 — напряму в браузері:**
```
відкрити index.html
```

**Варіант 2 — локальний сервер (рекомендовано):**
```bash
python3 -m http.server 8765
```
Потім перейти на `http://localhost:8765`
